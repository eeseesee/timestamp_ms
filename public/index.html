API Timestamp Microservice

Full stack Javascript app built with node.js, express.js, and Bootstrap by ECC

User stories:
    1) I can pass a string as a parameter, and it will check to see whether that string contains either a unix timestamp or a natural language date (example: January 1, 2016)
    2) If it does, it returns both the Unix timestamp and the natural language form of that date.
    3) If it does not contain a natural language date or Unix timestamp, it returns null for those properties.

Example usage:

https://APPNAME.herokuapp.com/February%2015,%202016
https://APPNAME.herokuapp.com/1455494400

Example output:

{ "unix": 1455494400, "natural": "February 15, 2016" }
